# 2026-Capstone

> 2026 캡스톤 프로젝트 및 실습 자료를 정리한 저장소입니다.  
> 프로젝트 기획안, 스킬 테스트, 발표용 자료 제작 관련 파일을 포함합니다.

---

## 프로젝트 개요

이 저장소는 2026 캡스톤 프로젝트 진행 과정에서 작성한  
기획 문서, 실습 파일, 테스트 자료를 체계적으로 정리하기 위해 생성되었습니다.

현재 포함된 내용:
- 프로젝트 아이디어 제안서
- presentation-generator.skill
- 클로드 스킬 테스트 자료

---

## 목표

- 캡스톤 프로젝트 관련 자료를 한 곳에서 관리
- 실험 및 테스트 과정을 기록
- 발표 및 문서 제작 흐름 정리
- 추후 결과물 확장 및 버전 관리
